1. 資料夾配置
image = rawImage: 有被辨識到照片的原始檔 + unObjectImage: 沒有被辨識到照片的原始檔
objectImage: 有被辨識到照片的辨識後圖片
json: 車道辨識後用以儲存車道線資訊的資料夾
DetectInfo_yyyy_mm_dd.txt: 裡面會包含拍攝時間、車號、辨識前照片名、辨識後照片名、破壞長度、破壞寬度、破壞面積、程度、經度、緯度、破壞種類、[左上x座標, 左上y座標, 右上x座標, 右上y座標]及標案號碼

2. 軟體功能簡介
A. 上方資訊區
路徑來源: 顯示目前顯示圖片的電腦路徑位置

圖名: 下拉式選單，可以改變目前顯示的圖片

道路類型: 有市區道路和省縣鄉道、隧道、橋樑兩種，選市區道路就可以填寫市區道路格子內的資訊，反之亦然
!!!出報告時必須要將舊的excel表格關閉，並且需選取道路類型並填寫格子內的資訊，不可留空

車道線數量: 可選擇車道線數量

破壞資訊表: 表格內會顯示此張照片被辨識到的資訊，依序是破壞類型、破壞程度、線段長、線段寬、面積和所在的車道，最底為計算出的各車道的PCI值
(車道會在畫了車道線，並且按下儲存後會再次計算並改變)

樁號: 可自行設定起始樁號和間隔，在excel表格內的A欄-里程(公里)就是使用樁號去計算，H欄的樁號也是
例如起始樁號為0、間隔20就會變成 -> A欄: 0k_020，H欄0.02

車道線: 車道線數量改變後即可多畫幾條車道線

車道寬度/長度: 每張照片的車道寬和長度，用來計算PCI值

B. 下方顯示區
圖片: 圖片會顯示出已辨識後的照片或是沒有偵測到破壞的照片原始檔，可以點選車道來畫車道線

右方欄位為人工計數區，在項目上點左鍵增加數字，點右鍵減少數字
!!!人工計數區內的數字不會被記錄，重讀資料夾後就會遺失

畫圖: 在圖片內點下滑鼠左鍵開始畫線、點下滑鼠右鍵停止畫線

C. excel報告
資料: 包含"有辨識到破壞的照片"各種資訊，依序是---
里程(公里)	圖片名	原始圖片                        	點圖                                 	經緯度	樁號	車道	PCI	破壞類型	破壞程度	破壞長度	破壞寬度	破壞面積

PCI統計: 統計所有照片的PCI，並繪有一直方圖

辨識統計: 所有照片中破壞數量的統計表，並繪有一原餅圖

3. 快捷鍵
F1~F4: 切換車道線
ctrl+S: 儲存當前圖片的畫出的車道線到json資料夾中
ctrl+C: 按下後即可使用方向鍵"左"、"右"來切換上一張或下一張圖片
ctrl+Z: 刪除目前選取的車道線
"+": 放大圖片
"-": 縮小圖片
